关于我和这个网站

Jun 23, 2017 in About

上次更新：2020-03-31

这个网站是基于 GitHub Pages 和 Hexo 的，所有的照片视频储存在 Amazon S3 和 OneDrive，其他图片等储存在 GitHub。虽然都有 CDN 分发，但是在朝鲜和其他地区仍可能访问缓慢。此外，会有 YouTube、GoogleMaps、Flickr 之类的链接，上述地区需要代理。

之前的内容有时候会更新，但是这个网站没有一个自动 tag 最后修改日期的功能，我暂时也没打算手动加上。

我已經盡可能地縮減在 GitHub 上儲存的包括背景圖片在內的所有圖片的文件大小了，並且幾乎所有字體和第三方 CSS 都使用 jsdelivr 分發了，但是我試了下在一些地區和條件下仍有時訪問緩慢，TTFB 巨長，無能為力。而且 Safari 非常執著於將所有元素全部載入才顯示頁面，Chrome 會覺得背景圖可以慢慢載入先把其他顯示出來。。

apple-touch-icon 来自 Pixiv / 卯月つくよ

這個 Blog 之後我應該會使用繁體漢字寫一些內容，反正我用拼音，也沒什麼本質差異。我倒是覺得繁體好看一點，當然，對於筆畫特別多的字，我也會採用合適的簡化字（主要是異體字兼簡體字）替換，不然太小影響閱讀。文字嘛，注重內容，少點「規範」和優越感。另外，我喜歡傳統明體的寫法，所以我會用日語的思源宋體作為首選。

2019-05-26
今年想多出去走走，下个月会去 🇩🇪，9 月底会去台北，国庆未知，感觉没什么 💴 了，很惨。想看看秋天的京都和冬天的北海道。其实还有挺多想去的地方的，随便列一下吧，看着就会是有钱的时候没时间、有时间的时候没钱了：冰岛（想看极光草帽山）、夏威夷（想看 Mauna Kea）、瑞士（想去冬季的马特洪峰和玩春季的山地过山车）、英国（没啥特别想去的地方，如果非要说就是中部地区图灵破解 Enigma 的那个博物馆）、新疆（果子沟好像很有意思）、阿根廷/智利（巴塔哥尼亚的秋）、美国黄石和冰川国家公园（Ansel Adams 在 Snake River 那的神作）、新西兰（感觉羊驼好多，🐏很有意思）……

网站更新：

2020-03-31
更新了一票东西，比如加入了语言自定，支持了繁体字体，尽量改为 CDN 分发资源，还解决好几个 bug。总之把一年前遗留的问题解决了，现在应该没有已知问题了。

2019-05-26
把图片搬迁到 OneDrive 了，还是寻求一个免费吧。。不过我用了一个 json 存下所有的链接对应的文件名，以后要是有更好的方案了方便写个脚本直接迁移。

2019-04-05
最近进入了一个新的阶段，然后上周末和今天花了三天来彻底把这个网站改成了我之前希望的样子，加上了诸如页内跳转的动画、思源宋体支持、懒加载图片、简化的回到目录按钮等等，还有一些排版上的比如对齐、字符段落间距、字号等等。其实我还是非常关注设计美学的，尽管我的「专业」跟这个毫不相关，当然如何让网页看上去美观也是很重要的事。
遗憾就是，我觉得 2019 的网页应该是会有一些载入动画和加载提示的，以及应该使用 PJAX。但是我目前都还没有好好系统学过 JavaScript，尝试了挺久，很多东西还是没能力实现，就此作罢，已经花了不少时间。
